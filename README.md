# NewsBot Network

AI-powered news website with daily auto-posting for Google AdSense monetization.

## Features
- Daily AI-generated news articles
- SEO optimized for search engines
- AdSense-ready layout
- Automated deployment via Vercel
- Responsive design with Tailwind CSS

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI/Claude API for content generation
- GitHub Actions for automation
- Vercel for hosting

## Setup
1. Clone this repository
2. Run `npm install`
3. Set up environment variables
4. Run `npm run dev`

## AdSense Integration
- Header banner ad slot
- In-article ad placements
- Sidebar ad units
- Footer ad space

## Daily Automation
Articles are automatically generated and published daily via GitHub Actions.
